# Collectathon

Una app web en Ionic para la gestion de colecciones audiovisuales.

Este proyecto consiste en el desarrollo de una aplicación móvil para la gestión de coleccionismo, particularmente de elementos de medios audiovisuales con una temática, narrativau objetivo. Estos ítems o elementos se caracterizan por ser obras con el fin de entretener o educar a su audiencia, como lo son los libros, películas, series de televisión, álbumes musicales, cómics, videojuegos, etcétera.

Al contrario que muchas aplicaciones y bases de datos en el mercado, el objetivo principal es aunar todos esos tipos de elementos en una misma app, con cada tipo previamente e implementado en el sistema con todos sus atributos posibles, pero que para los usuarios sean solo categorías y estos objetos reciban el mismo trato de manera transparente paralos usuarios.

Cada uno de estos ítems puede tener varias versiones diferentes que difieran en formato, año de salida, idioma, plataforma, edición especial, etc. Igualmente, también pueden salir fraccionados de manera serial en diferentes entregas de la misma historia como tomos, volúmenes o temporadas. Todas estas distintas salidas del mismo ítem se recogen bajo elnombre de ediciones. Los usuarios pueden crear, editar y eliminar no sólo el ítem padre sino las ediciones de éste. En realidad, el usuario añade o borra a sus listados y su espaciode ítems a las ediciones concretas, ya que una persona puede tener varias en la vida real.

Otra característica fundamental del sistema desarrollado es la posibilidad de crear y gestionar colecciones, que son listados de ediciones propias del usuario en las que puedeañadirlas y sacarlas a su gusto. Una colección es una forma de clasificar estas obras, comouna estantería o una caja que recoge elementos de una misma temática o característicasa elegir por el usuario. Un usuario puede tener cuantas colecciones guste, además de una lista de deseos para ediciones que no posee pero le gustaría tener.

Finalmente, la aplicación móvil desarrollada permite el filtrado de estos objetos por varios parámetros, como tipo de ítem, géneros, series, personas involucradas en su creación, o búsqueda de otros parámetros relevantes como título o campos particulares de un tipo.

# Requisitos de ejecución

Se requiere un dispositivo con sistema operativo Android/EMUI. La versión mínima de Android es 5.1 y la versión mayor es 9.1.0.

# Instalación de la app en dispositivos

Descargar el fichero collectathon-v1.05.apk del directorio raíz. Asegurarse antes que en Ajustes > Seguridad > Orígenes desconocidos esté activado. Abrir fichero y aceptar la instalación.
